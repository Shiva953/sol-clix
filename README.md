# sol-clix
A terminal based solana cli wallet.

## Trying it
To run it locally,

1. Clone the repository
   `git clone https://github.com/Shiva953/sol-clix`

2. cd into the repository
   `cd sol-clix`

3. Run it using `cargo run`
